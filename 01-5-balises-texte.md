# 01.5 — Balises texte essentielles  
  
## Explication  
  
| Balise | Utilisation |  
|------|------|  
| p | paragraphe |  
| strong | importance |  
| em | emphase |  
| ul | liste |  
| ol | liste ordonnée |  
| li | élément de liste |  
  
## Code  
  
```html  
<p>Ceci est un paragraphe.</p>  
<strong>Texte important</strong>  
<em>Texte en emphase</em>  
  
<ul>  
  <li>élément 1</li>  
  <li>élément 2</li>  
</ul>

  
```
