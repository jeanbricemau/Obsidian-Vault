# 01.2 — Structure principale d’une page  
  
## Explication  
Une page web moderne utilise des **balises sémantiques** pour structurer son contenu.  
  
Balises principales :  
  
| Balise | Rôle |  
|------|------|  
| header | en-tête du site |  
| nav | navigation |  
| main | contenu principal |  
| footer | pied de page |  
  
Bonne pratique : **un seul `main` par page**.  
  
## Code  
```html  
<body>  
  <header></header>  
  <nav></nav>  
  <main></main>  
  <footer></footer>  
</body>

  
`
