# 01.4 — Les titres HTML  
  
## Explication  
Les titres structurent la hiérarchie :  
  
- `h1` : titre principal  
- `h2` : sections  
- `h3` : sous-sections  
  
Règles :  
- éviter de sauter des niveaux  
- garder une hiérarchie logique  
  
## Code  
  
```html  
<h1>Titre principal</h1>  
<h2>Section</h2>  
<h3>Sous-section</h3>
```
