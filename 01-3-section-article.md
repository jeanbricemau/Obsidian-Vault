# 01.3 — Organiser le contenu avec section et article  
  
## Explication  
- `section` : bloc thématique (produits, articles, témoignages…)  
- `article` : contenu indépendant (post, card produit, news…)  
  
## Code  
  
```html  
<main>  
  <section>  
    <h2>Actualités</h2>  
  
    <article>  
      <h3>Nouveau produit</h3>  
      <p>Description du produit.</p>  
    </article>  
  
    <article>  
      <h3>Annonce importante</h3>  
      <p>Contenu de l'annonce.</p>  
    </article>  
  </section>  
</main>

  
```
