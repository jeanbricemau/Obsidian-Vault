# 01.1 — Le squelette minimal d’une page HTML

## Explication
- `<!DOCTYPE html>` indique que le document utilise **HTML5**
- `<html>` est la racine du document
- `<head>` contient les **métadonnées**
- `<body>` contient **le contenu visible**

## Code

```html
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Ma page</title>
</head>
<body>
</body>
</html>
